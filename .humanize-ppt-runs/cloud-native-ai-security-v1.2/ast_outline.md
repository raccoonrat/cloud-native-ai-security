# AST Outline

## Audience
Creators, product builders, and independent developers interested in AI tools and agent workflows.

## State
- Initial: 信息分散，缺少可讲路径。
- Desired: 形成清晰判断，并能执行下一步。

## Transfer
- S01 / hook: 抓住注意力：先把观众从信息疲劳里拉出来。
- S02 / context: 建立共同背景：说明为什么现在要听这件事。
- S03 / tension: 制造认知张力：指出旧理解和真实问题之间的差距。
- S04 / method: 给出方法：把复杂信息变成可执行路径。
- S05 / proof: 给出证据：用案例、步骤或指标证明它不是口号。
- S06 / takeaway: 收束行动：让观众带走一句可复述的方法。
